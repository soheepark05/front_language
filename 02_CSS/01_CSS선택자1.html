css 기본선택자

선택자란?
특정 html 요소를 선택할 때 사용하는 기능으로, 요소(=태그)를 선택하여 원하는 스타일과 기능을 적용할 수 있다.

1. 전체 선택자(*)

html 문서안의 모든 태그를 선택할 때 전체 선택자를 사용합니다.

        * {
            스타일 속성: 값;
            스타일 속성: 값;
            ...
        }

2. 태그 선택자(태그명)

html 문서내에 같은 태그를 모두 선택할 때 사용합니다.

        태그명 {
            스타일 속성: 값;
            스타일 속성: 값;
            ...
        }

3. 아이디 선택자(#아이디)

html문서 내에 해당하는 아이디의 속성값을 가진 태그만 선택할 때 사용한다. 

        #아이디 {
            스타일 속성: 값;
            스타일 속성: 값;
            ...
        }

클래스 선택자(.클래스)

html문서 내에 여러개의 태그를 동일한 클래스로 지정하여 선택할 때 사용한다. 

        .클래스 {
            스타일 속성: 값;
            스타일 속성: 값;
            ...
        }
